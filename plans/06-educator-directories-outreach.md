## Key Messaging Points

Emphasize for educator audience:
1. **Privacy-first**: Files never leave device (FERPA-friendly)
2. **Works on Chromebooks**: No install required
3. **Free**: No budget approval needed
4. **Fast**: 2-3 minute workflow
